El juego

Neon Lane Splitter es un arcade de carreras infinito con vista aérea y estética cyberpunk. El objetivo es simple: esquiva los coches enemigos cambiando de carril el mayor tiempo posible. Cada 10 segundos la velocidad aumenta un escalón — la dificultad no tiene techo.

El juego funciona a velocidad constante en cualquier ordenador, tablet o móvil, sin importar los fotogramas por segundo del dispositivo. Esto se consigue mediante un sistema de físicas basado en tiempo real (delta time): a 30 FPS o a 144 FPS, los coches se mueven exactamente igual.

En modo 2 jugadores, ambos jugadores compiten en el mismo dispositivo al mismo tiempo. Gana quien sobreviva más tiempo. Además, en este modo aparecen ítems especiales en la carretera — diamantes de colores que otorgan poderes temporales para desestabilizar al rival.

El origen

Neon Lane Splitter nació como un experimento personal: ¿hasta dónde se puede llevar un juego de navegador usando solo las APIs nativas del navegador? Sin Phaser, sin Unity, sin frameworks. Solo el Canvas de HTML5 y JavaScript vanilla.

Lo que empezó como una prueba de concepto de fin de semana acabó convirtiéndose en un arcade completo con modo para dos jugadores, dificultad progresiva, sistema de ítems y buff, récords guardados en cookies, partículas y efectos visuales de neón. La estética cyberpunk surgió de forma natural con el primer boceto — y se fue puliendo hasta el resultado final.

Es el primer juego publicado de este proyecto. Hay más ideas en camino.

Tecnología utilizada

▸HTML5 Canvas API
▸JavaScript ES6+
▸CSS3
▸Google Analytics
▸Google AdSense
▸GitHub Pages
▸Pointer Events API
▸requestAnimationFrame
▸Cookie API

Sin dependencias externas de juego. Todo el motor de físicas, la generación procedural de obstáculos, el sistema de partículas, los efectos de glow y la interfaz están escritos desde cero en menos de 1100 líneas de JavaScript.

El sistema de controles táctiles usa la Pointer Events API, que funciona tanto con touch como con ratón, y soporta múltiples toques simultáneos (multi-touch) para que ambos jugadores puedan pulsar sus botones a la vez sin interferencias.
